ⓘThis section displays an arbitrary path from one of the initial packages to the selected package. Click the break button so see how your dependencies would change if you were to remove a single edge. Click break all to remove all inbound edges to a package, removing it from the graph. This may be useful for removing distracting packages that you don't plan to eliminate. The bold nodes are dominators: nodes that are found on every path to the selected node. One way to break a dependency on a package is to break all dependencies on any of its dominators.
